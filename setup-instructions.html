🌐 Aveshe Energy Website — Setup Instructions

This guide sets up your website so your client can edit content (text, phone numbers, services, etc.) through a simple browser dashboard — no coding needed. Everything is 100% free and you keep full ownership.

⏱ Total time: About 30–45 minutes. You only do this once.

STEP 1 — Create a Free GitHub Account

GitHub stores your website files for free. Netlify and the CMS both connect to it.

1. Go to https://github.com and click Sign up
2. Choose a username (e.g. aveshe-energy), enter email and password
3. Verify your email address
4. Click New repository (green button or + icon top right)
5. Name it: aveshe-website
6. Set to Public
7. Tick "Add a README file"
8. Click Create repository

Upload your files to GitHub

1. Inside your new repository, click Add file → Upload files
2. Drag and drop ALL the files and folders from your ZIP into the upload area
   (Make sure admin/, _data/, images/ folders are included)
3. Scroll down and click Commit changes
4. ✅ Your files are now on GitHub
💡 If drag-and-drop doesn't upload folders, use GitHub Desktop app (free) — download from desktop.github.com

STEP 2 — Deploy on Netlify (Free Hosting)

1. Go to https://netlify.com and click Sign up
2. Choose "Sign up with GitHub" — this links them together
3. Once logged in, click Add new site → Import an existing project
4. Click GitHub
5. Select your aveshe-website repository
6. Leave all settings as default
7. Click Deploy site
8. Wait 1–2 minutes — Netlify gives you a URL like aveshe-energy.netlify.app
9. ✅ Your website is now live!

Set a custom site name (optional)

In Netlify: Site configuration → Site details → Change site name
Change it to something like aveshe-energy so the URL is aveshe-energy.netlify.app

STEP 3 — Enable the CMS Editor (Decap CMS)

This gives your client their login dashboard at yoursite.netlify.app/admin

3a — Enable Netlify Identity

1. In Netlify dashboard, go to Site configuration → Identity
2. Click Enable Identity
3. Scroll to Registration preferences → set to "Invite only"
   (So only people YOU invite can log in — your client can't be accessed by strangers)
4. Scroll to Git Gateway → click Enable Git Gateway
5. ✅ Identity is now enabled

3b — Invite Your Client

1. Still in Identity, click Invite users
2. Enter your client's email address
3. They receive an email — they click the link and set their own password
4. ✅ They can now log in at yoursite.netlify.app/admin

STEP 4 — Test the Editor

1. Go to yoursite.netlify.app/admin
2. Log in with your own Netlify account or your client's new account
3. You'll see the Aveshe Energy Content Editor dashboard
4. Click Site Settings → Hero Section
5. Change the headline text and click Publish
6. Wait 1–2 minutes and refresh your website — the change is live! ✅

STEP 5 — Connect Your Custom Domain (Optional)

If you want www.aveshe.com instead of aveshe-energy.netlify.app:

1. In Netlify: Domain management → Add a domain
2. Type your domain (e.g. aveshe.com)
3. Netlify will show you DNS records to add
4. Log in to wherever you bought your domain (GoDaddy, Namecheap etc.)
5. Add the DNS records Netlify shows you
6. Wait up to 24 hours — your custom domain will be live with free HTTPS ✅
